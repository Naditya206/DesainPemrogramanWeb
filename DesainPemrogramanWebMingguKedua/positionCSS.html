Position static

Position static digunakan mengatur element menjadi statis secara default. Elemen akan mengikuti posisi normal secara default. elemen tidak dipengaruhi oleh properti top, bottom, left dan right,:

Elemen ini diatur menggunakan posisi static;




position: relative;

Sebuah element html yang menggunakan position relative akan terletak pada posisi normal. mengatur properti atas, kanan bawah dan kiri dari elemn yang dipolisikan degan relatif. akan membuatnya semakin jauh dari posisi normalnya

Element ini diatur menggunakan posisi : relative;




position:fixed;

sebuah element html yang di setting dengan position ficed akan memiliki sifat tetap tanpa ada perubahan bahkan jika halaman website di scroll. berlaku pengaturan left bottom top dan right pada position fixed

Elemen ini diatur menggunakan posisi:fixed;















position absolute

Sebuah element html yang menggunakan position absolute akan diposisikan relatif dengan element yang lain mendahuluinya yang terdekat bukan relatif terhadap layar secara normal

Elemen ini diatur menggunakan posisi: relatif
element ini diatur menggunakan posisi absolute
